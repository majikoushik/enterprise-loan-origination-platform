Epic 9 Goal:

Prepare the project for Azure deployment using a professional, cloud-native architecture blueprint.

This epic should demonstrate Azure solution architecture thinking, secure configuration, infrastructure planning, deployment documentation, environment separation, and enterprise cloud-readiness.

Do not require actual Azure deployment to be completed unless the repository is already configured for it. The main goal is a complete, professional Azure deployment foundation.

Target Azure Architecture:

Use the following Azure-aligned deployment model unless the existing repository has a cleaner documented approach:

1. Angular frontend:

   * Azure Static Web Apps
     or
   * Azure Storage Static Website with CDN readiness

2. Backend APIs and workers:

   * Azure Container Apps

3. Container registry:

   * Azure Container Registry

4. Database:

   * Azure SQL Database

5. Messaging:

   * Azure Service Bus

6. Secrets and configuration:

   * Azure Key Vault
   * Managed Identity readiness

7. Monitoring:

   * Azure Application Insights
   * Azure Log Analytics
   * Azure Monitor readiness

8. Optional future edge/API layer:

   * Azure API Management
   * Azure Front Door


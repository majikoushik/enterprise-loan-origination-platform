Loan application fields:

* Customer ID
* Loan type
* Requested amount
* Requested tenure in months
* Loan purpose
* Declared monthly income
* Existing EMI obligations

Supported loan types for MVP:

* PersonalLoan
* HomeLoan
* VehicleLoan
* EducationLoan

Supported application statuses:

* Draft
* Submitted
* UnderReview
* EligibilityPassed
* EligibilityFailed
* Approved
* Rejected
* Cancelled

Validation rules:

* Customer ID is required.
* Customer ID must refer to an existing customer if customer lookup is available.
* Loan type is required.
* Requested amount must be greater than zero.
* Requested tenure must be between 6 and 84 months.
* Loan purpose is required.
* Declared monthly income must be greater than zero.
* Existing EMI obligations cannot be negative.
* Existing EMI obligations should not be greater than declared monthly income.

* `GET /api/v1/loan-applications`
